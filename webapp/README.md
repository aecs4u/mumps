# BLAS Benchmark Dashboard

This FastAPI app renders the latest BLAS benchmark results generated by:

- `make bench-blas`
- `make bench-blas-dense`
- `make bench-all-db` (runs both and ingests SQLite)
- `make bench-mkl-amd-vitis-db` (runs MKL vs AMD Vitis and ingests SQLite)

## Run

1. Install dependencies:

```bash
python -m pip install -r webapp/requirements.txt
```

2. Generate benchmark results:

```bash
make bench-blas
make bench-blas-dense
# Optional: run both and ingest SQLite in one command
make bench-all-db
```

3. Start the web app:

```bash
uvicorn webapp.main:app --reload
```

4. Open:

`http://127.0.0.1:8000`

## Notes

- The app reads files from `benchmarks/results` by default.
- The app prefers SQLite at `benchmarks/results/blas_benchmarks.sqlite` when available.
- You can override this path with `BLAS_BENCH_RESULTS_DIR`.
- You can override DB path with `BLAS_BENCH_DB`.
